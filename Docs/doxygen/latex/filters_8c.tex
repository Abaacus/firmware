Module containing I\+IR filter sample implementation. 

\begin{DoxyAuthor}{Author}
Richard Matthews
\end{DoxyAuthor}
Sample implementation of an I\+IR filter using the arm cmsis filter library. Could be used to replace simple low pass filters implemented elsewhere in the code, but is currently unused. The coefficients can be generated by matlab scripts or other sources
