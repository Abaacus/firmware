\doxysubsection{Detailed Description}
Board support package (B\+SP) header file. 

\begin{DoxyAuthor}{Author}
Richard Matthews
\end{DoxyAuthor}
This header file is a simple B\+SP, specifying which hardware to use on different build targets. Instead of directly refering to hardware in source code, create a define here and use that, then ensure the define exists for each build target it applies to.
