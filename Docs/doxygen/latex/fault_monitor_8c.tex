\doxysubsection{Detailed Description}
Module monitoring status of IL and H\+V\+IL. 

\begin{DoxyAuthor}{Author}
Richard Matthews
\end{DoxyAuthor}
Contains fault monitor task which monitors the status of the interlock loop (IL) and high voltage interlock loop (H\+V\+IL) and reports an error when either breaks. Also contains functions for getting the status of various points along the IL.
