Detailed Description

Module for communicating with ltc6811 chips on the AMS boards.

Author
Richard Matthews

The ltc6811 chips are battery monitoring chips, with one on each AMS board connected to 12 battery cells. The chips measure the voltage and temperature of the cells. They communicate with the BMU over isoSPI (an electrically isolated SPI interface). This file implements a driver for communicating with the ltc6811 chips
