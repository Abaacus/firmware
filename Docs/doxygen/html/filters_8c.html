Detailed Description

Module containing IIR filter sample implementation.

Author
Richard Matthews

Sample implementation of an IIR filter using the arm cmsis filter library. Could be used to replace simple low pass filters implemented elsewhere in the code, but is currently unused. The coefficients can be generated by matlab scripts or other sources
